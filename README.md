## 使用方法：

```shell
# 克隆项目
git clone https://github.com/zxpsuper/createVue.git
# 进入文件夹
cd createVue

# 安装依赖
npm install

# 进入开发
npm start

# 打包
npm run build

# git commit
npm run commit
```
